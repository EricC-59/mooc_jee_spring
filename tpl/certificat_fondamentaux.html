Java EE Spring ////
Prêt à l'emploi

Certificat

Vous avez acquis les fondamentaux du développement web en Java EE.

Servlet

  • Vous savez mettre en oeuvre le modèle de développement client-serveur à l'aide de l'API Servlet.
  • Vous maitriser la réalisation de formulaire html pour transmettre des données à une servlet java.
  • Vous savez manipuler les paramètre HTTP afin de générer des pages web dynamiques.
  • Vous avez appris les bases du protocole HTTP et savez réaliser des traitements dissociés en POST ou en GET.
  • Vous êtes en mesure d'utiliser une session pour conserver un état côté serveur entre deux requêtes d'un même utilisateur.

JSP

  • Vous savez générer des pages html dynamiques à l'aide de fichier JSP mélant du code java.
  • Vous utilisez les variables pré-définies pour accéder aux paramètres de requête et aux données de session.
  • Vous comprenez la séparation des préoccupations et dissociez les traitements de l'affichage à travers la mise en oeuvre du pattern MVC.

JDBC

  • Vous connaissez les opérations principales de l'API JDBC
  • Vous êtes capable de vous connecter à une base de données relationnelle à l'aide d'un Driver
  • Vous êtes sensibilisé à l'injection SQL et savez vous en prémunir en utilisant des requêtes préparées
  • Vous exécutez des requêtes de sélection ou de mise à jour de données
  • Vous comprenez l'intérêt du pattern DAO et êtes capable de réaliser des opérations "CRUD" sur des objets métiers
